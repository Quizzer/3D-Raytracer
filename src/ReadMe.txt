Übungsblatt 2

1. bewertete Agbae:	RayTracer

Aufruf des Skipts mit:
>> python Controller.py

Parametermitgabe:
>> python Controller.py [aufgaben-Nr] [Bildbreite] [Bildhöhe]

Beispielsweise Aufgabe 2 in der Größe 1920x1080 Pixel
>> python Controller.py 2 1920 1080

Defaultwerte bei Weglassen von einem der oben genannten Parameter sind:
>> python Controller.py 3 400 400


mitgelieferte Bilder zeigen:

RayTracer_aufg1 - Raytracer ohne rekusrive Strahlenverfolgung

Raytracer_aufg2 - Raytracer mit rekusriver Strahlenverfolgung

Raytracer_aufg3 - Raytracer mit rekusrive Strahlenverfolgung 
		und texturierter Grundfläche
